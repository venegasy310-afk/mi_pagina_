
# PROYECTO TURÍSTICO EN PYTHON FLASK

## INSTALACIÓN

1. Abrir terminal

2. Instalar Flask

pip install -r requirements.txt

3. Ejecutar proyecto

python app.py

4. Abrir navegador

http://127.0.0.1:5000

## TECNOLOGÍAS

- Python básico
- Flask
- HTML
- CSS
